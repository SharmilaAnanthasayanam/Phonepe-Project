# Phonepe Pulse Data Visualization and Exploration

### GOAL:
To create a Streamlit application that allows users to access and analyze Phonepe pulse data.

### Tools Used: 
* Streamlit
* MySQL
* Plotly
* mysql-connector-python
* Pandas
